# Noroff E-Com Store

A fully functional, responsive e-commerce application built as part of the **JavaScript Frameworks** course assignment at Noroff. The project demonstrates the use of React, TypeScript, and modern UI libraries to create a seamless shopping experience.

**[Live Demo Link](https://fed-online-shop.vercel.app)**

---

## What the project does

This application serves as a complete online shop where users can browse products, view details, and manage a shopping cart. Key features include:

* **Product Catalog:** Fetches and displays products from the Noroff API with real-time search and filtering.
* **Dynamic Pricing:** Automatically calculates and displays discount percentages and handles strike-through pricing for sales.
* **Detailed Views:** Individual product pages featuring descriptions, reviews, and tags.
* **Shopping Cart:** A fully functional cart allowing users to add/remove items and adjust quantities with a live total cost calculation.
* **Checkout Flow:** A multi-step process leading to a successful order confirmation and state reset.
* **Contact Form:** A validated contact page ensuring user input meets specific character and format requirements.

##  How users can get started

To run this project locally, follow these steps:

### Prerequisites

* Node.js (v18.0 or higher recommended)
* npm or yarn

### Installation

1. **Clone the repository:**
```bash
git clone https://github.com/NoroffFEU/jsfw-2025-v1-saimoh-jsf.git
cd jsfw-2025-v1-saimoh-jsf

```


2. **Install dependencies:**
```bash
npm install

```


3. **Run the development server:**
```bash
npm run dev

```


4. **Build for production:**
```bash
npm run build

```
